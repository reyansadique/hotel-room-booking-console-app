# 🏨 Hotel Room Booking Console App

A Java Console-Based Hotel Room Booking System developed to manage hotel room reservations efficiently using Object-Oriented Programming concepts.

## 🚀 Features
- Room Booking
- Booking Status Management
- Customer Information
- Room Availability Check
- Room Type Management
- Console-Based User Interface
- Object-Oriented Design

## 🛠️ Technologies Used
- Java
- Object-Oriented Programming (OOP)

See the screenshots uploaded in this repository.


## ▶️ How to Run
1. Download the project.
2. Compile:
   javac HotelApp.java
3. Run:
   java HotelApp
